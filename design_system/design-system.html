Motion & Interaction
Sidebar Transitions

The sidebar supports collapsed/expanded states via #menu-mini-button / #menu-expend-button. Toggling adds/removes a class on the body, and the sidebar transitions width with CSS. Mobile uses the .hamburger--arrowturn animation (#mobile-collapse).

Dropdown Animation

Dropdowns use Bootstrap's built-in transitions. The theme applies smooth opacity/transform transitions on .dropdown-menu and .nxl-h-dropdown.

Dark/Light Theme Toggle

Use the / toggle in the header to switch themes. All colors, backgrounds and borders transition smoothly between modes. The Theme Customizer (gear handle on the right) also exposes Navigation/Header/Skins/Typography options.

Counter Animation

Numeric KPIs use .counter spans animated by vendors.min.js on load.
